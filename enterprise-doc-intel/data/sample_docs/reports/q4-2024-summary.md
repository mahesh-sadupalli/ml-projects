# Q4 2024 Quarterly Business Summary

## Key Metrics
- Revenue: €42.3M (↑ 18% QoQ, ↑ 31% YoY)
- Active merchants: 12,450 (↑ 8% QoQ)
- Transaction volume: 28.7M transactions (↑ 22% QoQ)
- Average transaction value: €47.20
- System uptime: 99.97%
- Customer satisfaction (NPS): 72

## Strategic Highlights

### Product
- Launched Buy Now Pay Later (BNPL) feature across 4 markets (DE, NL, AT, BE)
- Released API v2 with improved webhook reliability and batch processing
- Mobile SDK adopted by 340 merchants (+120 QoQ)

### Engineering
- Migrated 60% of services to Kubernetes (target: 100% by Q2 2025)
- Reduced average API latency from 180ms to 95ms through caching optimization
- Completed SOC 2 Type II audit with zero critical findings
- Hired 15 engineers, bringing total engineering headcount to 180

### Market Expansion
- Entered UK market: 450 merchants onboarded in first 2 months
- Partnership with top 3 e-commerce platforms in DACH region
- B2B invoicing product pilot launched with 25 enterprise clients

## Financial Overview
- Gross margin: 68% (↑ 2pp vs Q3)
- Operating expenses: €28.1M
- EBITDA: €14.2M
- R&D spend: €11.5M (27% of revenue)

## Challenges
- Fraud losses increased 12% due to BNPL launch — additional ML models deployed in response
- Hiring in specialized roles (ML, security) remains competitive
- Regulatory changes in EU payment services directive require Q1 compliance updates

## Q1 2025 Priorities
1. Complete Kubernetes migration for all production services
2. Launch AI-powered fraud detection v2 (50% false positive reduction target)
3. Expand BNPL to France and Italy
4. Implement real-time settlement for premium merchants
5. ISO 27001 certification
